# 几何结构（测试说明）


几何结构测试主要包括：
* 几何结构的正确性；
* 复杂建模工具的正确性与运行效率。


## 测试目标
+ 几何体构造方式的**合理性**。
+ 几何体构造的**正确性**。
+ 几何体是否存在**自交**问题。
+ 几何体的**封闭性**。
+ 几何体在界面与求解器的行为是否具有**一致性**。


## 测试方法
+ 基本测试
  + 观察**几何体构造界面**，检测构造方式合理性。
  + 尝试各种**合理与不合理数值**作为输入，测试几何体的正确性。
  + 测试观察几何体是否存在**自交**问题。
  + 仿真测试几何体在界面的行为是否与求解器一致。
+ 功能点测试
  + 测试几何体**封闭性检测**功能，检测几何体的封闭性。
  + 测试**导出**功能，检测几何体能否正确导出 STL 格式。
  + 测试**导入**功能，检测工程文档能否正确导入 STL 格式模型。
+ 重点测试
  + 天线罩
  + 阵列
  + 垂直拉伸体
  + 多层物体
  + 旋转体
  + 强制网格面
