# 模型树窗口

模型树窗口显示当前活动的工程设计，该模型树包含了当前工程文档中所有的元件，以及它们之间的层级关系。


[image]("image/UI/other_wnd/modeltree.png", "模型树窗口")


操作说明：
* 通过该工程的模型树可以便捷的查看和操作(新建、删除、修改等)当前工程的**计算模式参数、模型参数、激励源参数、记录器参数及结果信息**；
* 通过点击工程树左侧的展开按钮和收起按钮按钮，可以**展开和收起**当前节点；
* 通过点击勾选按钮勾选和取消勾选，可以**显示和隐藏**对应节点的物体、激励源、记录器；在同一节点下，通过组合操作`shift键+鼠标左键`连续选择或`ctrl键+鼠标左键`加选多个对象。
