# 工作空间窗口

工程空间窗口默认位于 EastWave 7 工作界面左上方。通过主菜单**视图 > 工作空间**操作命令，可以显示/隐藏工作空间窗口。


[image]("image/UI/other_wnd/workspace.png", "工作空间窗口")


工作空间窗口通过树的形式对 EastWave 工程及计算结果进行管理，该窗口下主要包含以下内容：
* 所有已载入或曾经载入过的工程文件(`*.ewp`)；
* 计算结束后自动生成的与工程名称同名的的结果文件（`*.ewd`），双击结果文件即可跳转到后处理窗口对结果进行查看、处理。



```tips:warning-注意
在打开某些旧版文档时，EastWave 可能报警告称某些行内容无法识别，此时可以关闭该工程后，右键点击该工程文档并选择用文本文件方式打开，然后编辑相应行。
```
