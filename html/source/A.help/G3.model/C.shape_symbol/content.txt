# 几何元件
几何元件是 EastWave 7 提供的作为参数化对象的结构信息元件。       
    
> 使用 EastWave 几何元件功能可快速生成适合特定工程问题的物体模型，大大减小建模的复杂度。

```block:example-ewsl-script`endtag1
```ewsl
//添加相应的几何元件。
td.beg("cube","DShapeCube");
	td.set("unit","");
	td.set("opt_type#Cube#Shape",0x1);
	td.set("u3e_corner_1.u#Cube#Shape","0.0");
	td.set("u3e_corner_1.v#Cube#Shape","0.0");
	td.set("u3e_corner_1.n#Cube#Shape","0.0");
	td.set("u3e_corner_2.u#Cube#Shape","100.0");
	td.set("u3e_corner_2.v#Cube#Shape","100.0");
	td.set("u3e_corner_2.n#Cube#Shape","100.0");
td.end();

td.beg("topgroup1","DSymbolGrp");
	td.set("unit","");
	td.set("meshinfo","");
	td.set("material","");
	td.set("n_dp_level",0x0);
	td.set("transforms",[]);
	//把新构造的几何元件"cube"添加到该节点。
	td.set("children",["cube"]);
	td.set("reference","");
td.end();
```
