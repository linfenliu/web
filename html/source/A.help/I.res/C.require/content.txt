# 系统要求

## 基本配置：
* Microsoft Windows 2000/XP/Vista/Windows 7 32-bit or 64-bit；Linux；
* 采用x86/x86-64架构的 Intel/AMD CPU，主频不低于1GHz；
* 内存不低于512MB；
* 磁盘空间不低于1GB。

## 推荐配置：
* Microsoft Windows 7/8/10 64-bit；
* Intel Core i5/i7 处理器（四核以上，建议第四代酷睿核心以上），主频3GHz以上；
* 内存8GB以上，建议组建双通道或多通道；
* 应用程序空间不低于1GB，另有专用计算分区20GB以上。
* 配置512MB以上显存的独立显卡，可以显著提高操作大量几何单元的显示速度。

## 高性能计算服务器配置建议：
* Microsoft Windows Server 2008/2012 64-bit；
* Intel Xeon系列CPU，6核以上，主频3GHz以上；
* 双路以上服务器架构，建议四路或以上；
* 内存64GB以上，建议组成3通道或4通道；
* 使用大容量高速固态硬盘存放计算结果，可大大提高时域截面场的记录速度；
* 使用多达1TB以上的空间用于保存每次记录结果。
