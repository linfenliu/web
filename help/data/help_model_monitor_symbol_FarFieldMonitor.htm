时域远场监视器

频域远场监视器通过等效原理计算出电磁波在指定距离（按无限远近似处理）处的电磁波场的信号波形。

image 3: 频域远场监视器设置
image 4: 时间设置
时间设置包括：
  1. 自动设置时间步
  2. 按整数步设置（整数）
    • 起始
    • 结束
    • 步长
  3. 按物理时间步设置（浮点数，物理时间，文档时间单位）
    • 起始
    • 结束
    • 采样时间间隔
  4. 按频率宽度设置（浮点数，物理时间，文档时间单位、频率单位）
    • 起始
    • 结束
    • 频率宽度
  5. 设定时间序列（浮点数，物理时间，文档时间单位）
    • 设定时间序列
