所有材料元件

EastWave 中，材料作为元件进行定义。赋予了材料的几何元件成为物理元件。

材料属性

  • 显示属性
    • 颜色
    • 透明度
    • 反射方式
    • 绘制方式（线框模型、面模型、纹理）
  • 基本电属性
    • $\varepsilon$、$\sigma_e$
    • $\mu$、$\sigma_m$
  • 高级属性
    • 由材料的物理模型决定参数
    • 其他物理参数
    • 其他参数
